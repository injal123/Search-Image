# SearchGallery


## Description:
SearchGallery is an interactive image search web application built using HTML, CSS, and JavaScript.
It integrates with the Unsplash API to fetch and display high-quality images based on user-entered keywords.
This project demonstrates API integration, asynchronous JavaScript, and DOM manipulation skills.


## Tech Stacks:
- HTML5 
- CSS3 
- JavaScript (ES6) – For logic, API calls, and dynamic rendering
- Unsplash API – To fetch real-time image results
- Git & GitHub – Version control and repository management
- GitHub Pages – For live deployment


## How It Works:
1. Enter a keyword (e.g., “mountains”, “cars”, “technology”) into the search box.
2. Click the Search button to fetch related images from the Unsplash API.
3. Scroll through the results displayed in a responsive image grid.
4. Click Load More to fetch additional images for the same keyword.


## Features:
- Real-time image search using Unsplash API
- “Load More” functionality for continuous browsing
- Responsive and clean gallery layout
- Error handling for empty or invalid searches
- Smooth API-driven experience
